Demon souls item randomiser by Keeko

--- Installation Instructions --
You will need access to the files Demon Souls. Either by having the game files for emulation or by having
your console jailbroken so you can ftp the modded file.

BACKUP YOUR ORIGINAL GAMEPARAM FOLDER

Select options in the randomiser software and click randomise, this will generate a new file.

Drag this new file to <Path>
